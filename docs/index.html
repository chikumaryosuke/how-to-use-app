Bluetoothでセンサー値を遠隔取得しよう

Bluetoothモジュールとペアリングしよう

MooBtConnectアプリを開き、[Bluetooth検索]→[BT304]の順にクリックしていく。 ペアリング成功と表示されれば完了

Bluetoothモジュールに接続しよう

BTConnectionアプリを開いて[ARDUINO MODE]か[C-CUBIC MODE]を自分に合ったモードを選び、クリックする。 次に[CONECT]ボタンをクリックする。 [CONECT]ボタンの下にconeectedと表示されれば接続完了。 これで、対象の機体からBluetoothモジュールを通じて、センサーデータが送られてくる！
